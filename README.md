# Deepfake Guardian

A sophisticated AI-powered security system designed to detect and prevent deepfake-based unauthorized access. This project implements advanced liveness detection, face verification, and secure data handling.

## 🚀 Features

- **Liveness Detection**: Real-time analysis to distinguish between live humans and digital spoofs.
- **Face Verification**: High-precision face recognition and matching.
- **Secure Vault**: Encrypted storage for sensitive user data and captured evidence.
- **Blockchain Logs**: Immutable logging of authentication attempts for enhanced security.
- **Cyberpunk UI**: A modern, dark-themed dashboard with glassmorphism and smooth animations.

## 🛠️ Technology Stack

- **Backend**: Python, Flask
- **Security**: Advanced encryption and blockchain-inspired logging
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript
- **Detection Logic**: OpenCV, AI-based face detection

## 📂 Project Structure

- `backend/`: Core logic, authentication, database management, and security engines.
- `frontend/`: Templates and static assets for the user interface.

## 🚦 Getting Started

### Prerequisites
- Python 3.x
- Required libraries (see `requirements.txt` if available)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/IS-New.git
   ```
2. Navigate to the project directory:
   ```bash
   cd IS-New
   ```
3. Install dependencies (it's recommended to use a virtual environment):
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python backend/app.py
   ```

## 🔒 Security

This project prioritizes user privacy and data security. All captured liveness data is encrypted before storage, and authentication attempts are logged in a secure, tamper-proof format.
